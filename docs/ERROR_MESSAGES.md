# エラーメッセージ一覧 / Error Messages List

このドキュメントは、AIディベート拡張機能で表示される可能性のあるすべてのエラーメッセージと、それらが表示される条件を説明しています。

This document describes all error messages that may be displayed in the AI Debate Extension and the conditions under which they appear.

## 1. タブが見つからない場合 / When Tabs Are Not Found

### 同じAIタイプの場合（ChatGPT vs ChatGPT または Gemini vs Gemini）

#### タブが0個の場合:
```
❌ [AI名]のタブが見つかりません
必要な条件:
• [AI名]のタブを2つ開いてください
• 各タブでログインしていることを確認してください
• タブを開いた後、拡張機能アイコンをクリックしてください
```

#### タブが1個の場合:
```
❌ [AI名]のタブが1つしか開かれていません
必要な条件:
• [AI名]のタブをもう1つ開いてください（合計2つ必要）
• 両方のタブでログインしていることを確認してください
```

### 異なるAIタイプの場合（ChatGPT vs Gemini）

```
❌ 必要なAIタブが開かれていません: [不足しているAI名]
必要な条件:
• ChatGPTのタブを開いてください (chat.openai.com または chatgpt.com)
• Geminiのタブを開いてください (gemini.google.com)
• 各タブでログインしていることを確認してください
• タブを開いた後、拡張機能アイコンをクリックしてください
```

## 2. 入力検証エラー / Input Validation Errors

### ターン制限が範囲外:
```
⚠️ ターン制限は1から20の間で設定してください
```

### 遅延が範囲外:
```
⚠️ 遅延は1から30秒の間で設定してください
```

## 3. メッセージ送信エラー / Message Sending Errors

### 入力ボックスが見つからない（ChatGPT）:
```
❌ ChatGPTの入力ボックスが見つかりません
必要な条件:
• ChatGPTにログインしていることを確認してください
• 新しい会話を開始していることを確認してください
• ページを更新してから再試行してください
```

### 入力ボックスが見つからない（Gemini）:
```
❌ Geminiの入力ボックスが見つかりません
必要な条件:
• Geminiにログインしていることを確認してください
• 新しい会話を開始していることを確認してください
• ページを更新してから再試行してください
```

### 送信ボタンが見つからないか無効（ChatGPT）:
```
❌ ChatGPTの送信ボタンが見つからないか、無効になっています
必要な条件:
• メッセージが入力されていることを確認してください
• ChatGPTが応答中でないことを確認してください
• ページを更新してから再試行してください
```

### 送信ボタンが見つからないか無効（Gemini）:
```
❌ Geminiの送信ボタンが見つからないか、無効になっています
必要な条件:
• メッセージが入力されていることを確認してください
• Geminiが応答中でないことを確認してください
• ページを更新してから再試行してください
```

### メッセージ送信失敗:
```
❌ メッセージ送信エラー: メッセージの送信に失敗しました
必要な条件:
• [AI名]ページが完全に読み込まれていることを確認してください
• [AI名]にログインしていることを確認してください
• ページを更新してから再試行してください
```

## 4. 応答タイムアウト / Response Timeout

### ChatGPTの応答タイムアウト:
```
⚠️ ChatGPTの応答がタイムアウトしました
考えられる原因:
• 応答が非常に長い可能性があります
• ChatGPTがエラーを返した可能性があります
• ネットワークの問題がある可能性があります
対処方法:
• 遅延設定を増やしてください
• ページを更新してから再試行してください
```

### Geminiの応答タイムアウト:
```
⚠️ Geminiの応答がタイムアウトしました
考えられる原因:
• 応答が非常に長い可能性があります
• Geminiがエラーを返した可能性があります
• ネットワークの問題がある可能性があります
対処方法:
• 遅延設定を増やしてください
• ページを更新してから再試行してください
```

## 5. ディベート中のエラー / Errors During Debate

### 他のAIへの送信失敗:
```
❌ [AI名]へのメッセージ送信エラー: メッセージの送信に失敗しました
必要な条件:
• [AI名]ページが開いていることを確認してください
• [AI名]にログインしていることを確認してください
• ページを更新してから再試行してください
```

## 6. 情報メッセージ / Informational Messages

### トピックが指定されていない:
```
⚠️ トピックが指定されていません。手動で会話を開始してください。
```

または

```
ℹ️ トピックが指定されていないため、手動で会話を開始してください
```

## テストシナリオ / Test Scenarios

これらのエラーメッセージをテストするには:

1. **タブなしでテスト** - AIタブを開かずに拡張機能を起動
2. **1つのタブのみでテスト** - 同じAIのディベートで1つのタブのみ開く
3. **ログアウト状態でテスト** - AIサービスからログアウトしてメッセージを送信
4. **無効な入力でテスト** - ターン制限に0または30以上の値を入力
5. **タイムアウトテスト** - 非常に複雑な質問で長い応答をトリガー

## 改善点 / Improvements Made

1. **具体的なエラーメッセージ** - 何が問題なのかを明確に説明
2. **解決策の提示** - 各エラーで次に何をすべきかを示す
3. **日本語対応** - すべてのメッセージが日本語で表示
4. **段階的な詳細** - 簡潔なエラーメッセージと詳細な説明の両方を提供
5. **検証機能** - ディベート開始前に入力を検証
